How to use the Data Editor
1. Run program
2. Press "INITIALIZE" to load Data.dat
3. Press "Next Value"/"Prev Value" to find var to change
4. Change value by replacing the contents of the text box with different contents
5. Store changes to a value by pressing "Apply"
6. Repeat 3-5 until done.
7. Press "SAVE TO FILE" to save changes.
8. Quit at any time.
